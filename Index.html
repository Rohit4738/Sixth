T2

if Statement

Info

Executes a block only when the condition is true. If false, the block is skipped entirely.

if (condition) {
  // body
}
Code
#include <stdio.h>
int main() {
  int n = 10;
  if (n > 0)
    printf("Positive\n");
  return 0;
}
T2

if-else Statement

Info

Provides two paths — if block runs when true, else block runs when false. Exactly one block always executes.

if (cond) { ... }
else { ... }
Code
int n = 5;
if (n % 2 == 0)
  printf("Even\n");
else
  printf("Odd\n");
T2

Nested if-else

Info

An if-else placed inside another if-else. Used when decisions depend on previous conditions. Can get deep — keep it readable.

Code
int a=5, b=10, c=3;
if (a>b) {
  if (a>c) printf("a max\n");
  else    printf("c max\n");
} else {
  if (b>c) printf("b max\n");
  else    printf("c max\n");
}
T2

else-if Ladder

Info

Chain of conditions checked top-to-bottom. First true block executes and rest are skipped. Good for multiple mutually exclusive cases.

if (c1) {...}
else if (c2) {...}
else if (c3) {...}
else {...}
Code — Grade
int m = 75;
if (m >= 90)      printf("A\n");
else if (m >= 75) printf("B\n");
else if (m >= 60) printf("C\n");
else              printf("Fail\n");
T2

switch Statement

Info

Matches an integer/char expression against constant cases. Cleaner than long else-if ladders for discrete values. Needs break to prevent fall-through.

switch(expr) {
 case v: ...; break;
 default: ...;
}
Code
int day = 3;
switch(day) {
  case 1: printf("Mon\n"); break;
  case 2: printf("Tue\n"); break;
  case 3: printf("Wed\n"); break;
  default: printf("Other\n");
}

Q2

Factorial of a number using loop

C Code
#include <stdio.h>
int main() {
  int n, i;
  long fact = 1;
  printf("Enter n: ");
  scanf("%d", &n);
  for (i=1; i<=n; i++)
    fact *= i;
  printf("%d! = %ld\n", n, fact);
  return 0;
}
Pseudocode
  • START
  • INPUT n
  • SET fact = 1, i = 1
  • WHILE i ≤ n
  • fact = fact × i
  • i = i + 1
  • PRINT fact
  • STOP

Q3

Define loops — types, importance, syntax

Info + Syntax

A loop repeats a block of code until a condition fails. It avoids code repetition and handles bulk operations efficiently.

Types:

// for — known count
for(i=0; i<n; i++) { }

// while — condition first
while(cond) { }

// do-while — runs at least once
do { } while(cond);
Pseudocode (for loop)
  • START
  • SET counter = initial
  • CHECK condition
  • If true: execute body
  • Update counter
  • Go back to CHECK
  • If false: exit loop
  • STOP
Q4

Difference: switch-case vs if-else

Comparison

switch-case

• Works only with integers/chars

• Faster for many discrete values

• Needs break to stop fall-through


if-else

• Works with any condition/range

• More flexible (>, <, &&, ||)

• No fall-through issues

Code Comparison
// switch-case
switch(x) {
  case 1: printf("One"); break;
  case 2: printf("Two"); break;
}

// if-else equivalent
if (x == 1)      printf("One");
else if (x == 2) printf("Two");

T4 Q1

Function: Area of Circle → Volume of Cylinder (πr²h)

C Code
#include <stdio.h>
#define PI 3.14159

float circleArea(float r) {
  return PI * r * r;
}

float cylinderVol(float r, float h) {
  return circleArea(r) * h;
}

int main() {
  float r, h;
  printf("Enter r and h: ");
  scanf("%f %f", &r, &h);
  printf("Area   = %.2f\n", circleArea(r));
  printf("Volume = %.2f\n", cylinderVol(r, h));
  return 0;
}
Pseudocode
  • START
  • INPUT r, h
  • FUNCTION circleArea(r)
  • RETURN π × r²
  • FUNCTION cylinderVol(r, h)
  • RETURN circleArea(r) × h
  • CALL circleArea(r) → PRINT
  • CALL cylinderVol(r,h) → PRINT
  • STOP

T4 Q2

Function: Print all alphabets between 2 given alphabets

C Code
#include <stdio.h>

void printBetween(char a, char b) {
  char c;
  if (a > b) { // swap if needed
    c=a; a=b; b=c;
  }
  for (c=a; c<=b; c++)
    printf("%c ", c);
  printf("\n");
}

int main() {
  char x, y;
  printf("Enter 2 alphabets: ");
  scanf(" %c %c", &x, &y);
  printBetween(x, y);
  return 0;
}
Pseudocode
  • START
  • INPUT a, b
  • IF a > b: SWAP a, b
  • FUNCTION printBetween(a, b)
  • SET c = a
  • WHILE c ≤ b
  • PRINT c
  • c = next char
  • CALL printBetween(a, b)
  • STOP
